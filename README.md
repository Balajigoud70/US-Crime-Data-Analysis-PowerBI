# US-Crime-Data-Analysis-PowerBI
Interactive Power BI dashboard analyzing crime patterns and trends across various US states.
Objective: To identify crime patterns and trends across different states using data visualization.
Tech Stack: Power BI (DAX, Data Modeling), SQL, Excel.
Key Insights:

Analyzed crime distribution based on year and location.

Identified high-risk areas using a Star Schema model.

Created interactive dashboards for better decision-making.
